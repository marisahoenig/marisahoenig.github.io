# Marisa Personal Site

Adapted from <a href="https://github.com/rbhatia46/React-Portfolio">Rahul Bhatia</a>'s template. Many thanks!

## Pushing to Master
- Push to source
- Checkout master
- `git merge source`
- `git checkout source`
- `yarn deploy`

## Running locally
`npm install` - install node modules
`npm start` - start app on http://localhost:3000
